所见即所得——Typora使用技巧（基础篇）

发表于2021-03-12更新于2021-03-12字数统计897阅读时长5分阅读次数

​ Typora是一款支持实时预览的Markdown文本编辑器，支持且仅支持Markdown语法的文本编辑（ a truly minimal markdown editor )。而Markdown是一种用来编写结构化文本文档的纯文本格式，我们可以在双手不离开键盘的情况下，对文本进行一定程度的格式排版。当然，为了防止一些程序bug的发生，Typora保留了“源代码模式”，可通过快捷键**Ctrl+/**的方式进行查看。我之前都是用Sublime Text编辑md文档，自从开始使用Typora，编辑md文档的速率和舒适度都提高了很多。因此特地在此总结一下Typora编辑的一些逻辑和技巧。

​ 下面正式开始介绍Typora的一些基础使用技巧（包括标题、文字、代码、图片、表格等），本文章针对windows系统，mac系统上的操作大致类似。
