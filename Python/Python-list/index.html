Python学习-列表简介

发表于2021-06-11更新于2021-07-15字数统计705阅读时长5分阅读次数

列表由一系列按特定顺序排列的元素构成。下面记录几个列表的常用简单操作。

在列表中修改、添加或删除元素

添加元素

  1. **.append(元素)**直接在列表末尾添加元素

  2. **.insert(位置,元素)**在给定位置处插入给定元素

删除元素

  1. del 列表名[位置] 删除元素

    del语句删除的元素无法再访问

  2. **.pop(位置)**删除元素

pop()方法除了删除给定位置元素外，还可返回被删除元素（弹出栈顶元素）

  1. **.remove(元素)**删除元素

remove()方法只可删除第一个元素，若要删除的元素在列表中出现多次，可以使用循环全部删除。

组织列表

  1. **sort()**方法对列表永久排列

    sort方法后，列表内元素顺序永久变为排序后的

  2. 函数**sorted()**对列表临时排列

  3. 倒着打印列表

    用方法reverse反转列表元素的排列顺序

  4. 确定列表长度

    使用函数len()
